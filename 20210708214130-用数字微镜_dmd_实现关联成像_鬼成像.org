* 关联成像、鬼成像是什么
得鬼成像名是因为：像端不通过成像物体。
- 参考端（reference arm） :: 通过收集参考光收集空间信息，高空间分辨率探测器
- 像端（image arm） :: 收集信号光总强度信息，无空间分辨率探测器
- 信号光 :: 照射待成像物体，经衍射后传到像端的光
- 参考光 :: 与信号光传播相同距离，但在自由空间中传播的光
