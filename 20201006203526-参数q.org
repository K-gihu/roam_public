#+title: 参数q
#+roam_tags: 光学
#+roam_alias:

- 参数 q 和谐振腔有关，与[[file:20201006132210-基模高斯光束.org][曲率因子]] \(R\) ，[[file:20201006132210-基模高斯光束.org][束宽]] \(w\) 有关。
- \[q = Z - iZ_R\]
- 可以用来描述振幅 \(A\)
- 可以用输入平面的 q 参数来求输出平面的 q 参数（利用[[file:20201006204638-abcd矩阵.org][ABCD矩阵]]）：\(q_2 = \frac{Aq_1 + B}{Cq_1 + D}\)
